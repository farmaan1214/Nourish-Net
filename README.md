# NourishNet 🌾

An AI-powered Food Donation Platform designed to bridge the gap between surplus food and those in need. NourishNet connects donors (restaurants, events, individuals) with verified receivers (NGOs, shelters, community kitchens) to eliminate food waste efficiently.

## 🚀 Features

* **Role-Based Dashboards:** Dedicated interfaces for Donors, Receivers, and platform Admins.
* **Real-time Listings:** Donors can list surplus food with categories, quantities, expiry times, and pickup locations.
* **Seamless Claiming:** Verified NGOs can browse available donations and request pickups instantly.
* **Admin Verification:** Secure approval workflow ensuring only legitimate organizations can claim food.
* **Impact Tracking:** Donors can track their total meals served and earn donation certificates.
* **AI FAQ Chatbot:** A built-in virtual assistant to guide users through the donation and registration process.

## 🛠️ Tech Stack

* **Frontend:** HTML5, Vanilla CSS (Custom Design System), JavaScript
* **Backend:** Node.js, Express.js
* **Database:** SQLite (Persistent storage)
* **Deployment:** Pre-configured for Render.com

## ⚙️ How to Run Locally

1. Make sure you have [Node.js](https://nodejs.org/) installed.
2. Clone this repository or download the files.
3. Open a terminal in the project folder and install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000`

*Note: The platform automatically seeds demo data (users and donations) the first time you run it!*
